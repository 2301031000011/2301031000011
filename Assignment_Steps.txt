1. Initialize a Git repository (if not already done)
If you haven't already created a Git repository locally, follow these steps:

Open your terminal/command prompt.
Navigate to your assignment directory using cd command.
bash
Copy
cd path/to/your/assignment
Initialize the Git repository:
bash
Copy
git init
2. Add your files to the repository
After initializing, you need to stage your files for commit:

bash
Copy
git add .
This command adds all the files in the current directory to the Git staging area. If you only want to add specific files, replace the . with the file names.

3. Commit your files
Now, commit the added files with a message:

bash
Copy
git commit -m "Initial commit of my assignment"
4. Create a remote repository on GitHub (if not already created)
Go to GitHub and log in to your account.
Click the + icon at the top right and select New repository.
Name your repository (e.g., "my-assignment") and click Create repository.
5. Connect your local repository to GitHub
After creating the repository on GitHub, you will be provided with a URL (either HTTPS or SSH). To link your local Git repository to GitHub, run the following command:

bash
Copy
git remote add origin https://github.com/yourusername/your-repository-name.git
Make sure to replace yourusername and your-repository-name with your GitHub username and repository name.

6. Push your changes to GitHub
Now, push your local commits to the GitHub repository:

bash
Copy
git push -u origin master
If you're using a branch other than master (e.g., main), use that branch name instead.

7. Verify
Go to your GitHub repository in the browser, and you should see your assignment files uploaded.
